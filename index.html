Welcome to groundwork-jekyll

This is a starter framework for Jekyll sites.

Why would you use this?

  1. Fast to build
  2. Easily deploy and manage on Heroku servers
  3. No Database or CMS to maintain/update which means less long term support and risk
  4. Super stable HTML pages
  5. Incredibly extensible
  6. Easy editing interface with Prose

Some nerdy features include

  1. sass compiling and live reload on all changes
  2. full require backbone/underscore routed app compiled in to one built file
  3. server side jekyll build and public module loading, and js build
